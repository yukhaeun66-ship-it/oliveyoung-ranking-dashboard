## 🚀 실행 방법

### 1) 대시보드 바로 열기
`올리브영_랭킹_대시보드.html` 파일을 브라우저에서 열면 샘플 데이터로 즉시 실행됩니다.

### 2) 실제 데이터 수집 후 대시보드 연동

```bash
# 패키지 설치
pip install requests beautifulsoup4

# 스크래퍼 실행 → ranking_data.json 자동 생성
python oliveyoung_scraper.py
```

이후 대시보드에서 **[새로고침]** 버튼 클릭 → 실제 랭킹 데이터로 업데이트

### 3) 매일 자동 실행 (선택)
Windows 작업 스케줄러 또는 cron을 이용해 `oliveyoung_scraper.py`를 매일 1회 실행하면 7일치 히스토리가 자동 누적됩니다.

## 🎨 대시보드 주요 기능

### 1. 카테고리 탭 필터
- 전체 / 스킨케어 / 마스크팩 / 선케어 / 클렌징 / 메이크업 / 헤어 / 바디·헬스
- 탭 클릭 시 해당 카테고리 상품만 필터링

### 2. 통계 카드 (상단)
| 카드 | 내용 |
|------|------|
| 집계 상품 수 | 전체 랭킹 상품 수 |
| 신규 진입 | 이번 주 NEW 진입 상품 수 |
| 최고 상승 | 가장 많이 순위가 오른 브랜드 |
| 최고 하락 | 가장 많이 순위가 내린 브랜드 |

### 3. TOP 3 포디움
- 🥇🥈🥉 메달 표시
- **실제 올리브영 상품 썸네일 이미지** 표시
- 순위 변동 배지 (▲ 상승 / ▼ 하락 / NEW)
- **[🔗 바로가기]** 버튼 → 올리브영 상품 상세 페이지 바로 이동

### 4. 전체 랭킹 테이블
| 열 | 내용 |
|----|------|
| 순위 | 현재 랭킹 번호 (TOP 3는 초록색 강조) |
| 변동 | ▲N 상승 / ▼N 하락 / NEW / — 유지 |
| 상품 | 썸네일 + 브랜드명 + 상품명 + 카테고리 |
| 가격 | 현재 판매가 |
| 7일 추이 | 미니 스파크라인 차트 (상승=빨강, 하락=파랑) |
| 바로가기 | 올리브영 상품 상세 페이지 링크 버튼 |

### 5. TOP 10 랭킹 추이 차트
- 최근 7일간 TOP 10 상품의 순위 변화를 꺾은선 그래프로 시각화
- 각 상품 라인 개별 on/off 가능
- Chart.js 기반

## 🔍 스크래퍼 동작 원리

```
올리브영 랭킹 페이지 (getBestList.do)
        ↓ curl / requests
  HTML 파싱 (BeautifulSoup)
        ↓
  상품 정보 추출
  - goodsNo (상품 고유번호)
  - 썸네일 이미지 URL (image.oliveyoung.co.kr)
  - 브랜드명 (.tx_brand)
  - 상품명 (.tx_name)
  - 할인가 (.tx_cur .tx_num)
  - 상세 페이지 URL
        ↓
  ranking_history.json 에 날짜별 누적 저장
        ↓
  ranking_data.json 생성 (대시보드가 읽는 파일)
```

### 파싱 대상 HTML 선택자

```css
.prd_info              /* 상품 블록 */
.tx_brand              /* 브랜드명 */
.tx_name               /* 상품명 */
.tx_cur .tx_num        /* 할인 판매가 */
.thumb_flag.best       /* 순위 번호 */
img[src*="thumbnails"] /* 썸네일 이미지 */
a[href*="goodsNo"]     /* 상품 상세 페이지 링크 */
```
